# Aura Design Studio — Concept Project

A high-end, editorial-style web application developed for a boutique interior design and architecture studio. This project serves as a technical and aesthetic benchmark for luxury digital presence.

---

## 🏛️ Project Vision
The goal was to move away from generic templates and create a digital experience that mirrors the sophistication of a physical architectural space. The UI transitions between **Premium Dark** and **Minimalist Light** themes to evoke different emotional responses as the user explores the studio's philosophy and portfolio.

## 🛠️ Tech Stack
* **Framework:** Next.js 14 (App Router)
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion (Scroll-linked animations, complex transitions)
* **Typography:** Playfair Display (Serif) & Inter (Sans)
* **Deployment:** Vercel

## ✨ Key Features
* **Contextual UI:** Seamless theme switching based on section context.
* **Asymmetric Layout:** Modern, editorial-inspired grid system for project showcases.
* **Dynamic Portfolio:** High-resolution imagery optimization with Next/Image.
* **Fully Responsive:** Optimized "one-scroll" contact experience for mobile and desktop.
* **Performance:** Achieved near-perfect Lighthouse scores for SEO and accessibility.

## 🚀 Purpose
This is a **Concept Project** developed to showcase the intersection of high-end design and modern web performance. It is designed to be easily adaptable for premium brands looking for a unique digital identity.
